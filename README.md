### 🔧 VSMP — Vehicle Service Management Platform
> *The one that taught me what "production-ready" actually means*

A multi-role SaaS garage management system. Four dashboards, one backend.

```
Customer       → book services, track vehicles, view invoices
Mechanic       → manage assigned jobs, request parts, AI diagnosis help
Service Manager→ oversee operations, manage bookings
Admin          → full control: users, revenue, inventory, analytics
```

- JWT auth with `httpOnly` cookies, refresh token rotation, `tokenVersion` invalidation
- Gemini AI support chat widget + mechanic diagnosis assistant
- Docker + GitHub Actions CI/CD → Render

`React` `Node.js` `MongoDB` `Docker` `Gemini AI` `GitHub Actions`
